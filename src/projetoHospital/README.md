## O QUE FEZ?

- [ ] cadastrar pacientes e médicos.;
- [ ] informar quais os médicos que atendem determinado paciente;
- [ ] enviar mensagens de um paciente para seus médicos;
- [ ] enviar mensagens de um médicos para seus pacientes;
